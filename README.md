# Retail-Sales-Dashboard (Python Project)
-PROJECT OVERVIEW
This project focuses on analyzing a retail sales dataset to uncover meaningful business insights and visualize key performance indicators using Python.
The dashboard helps understand customer behavior, sales trends, and product performance through clear and interactive visualizations.

-TOOLS & TECHNOLOGIES USED
Python
Pandas – data cleaning & analysis
Matplotlib & Seaborn – data visualization
Jupyter Notebook

-DATASET DESCRIPTION
The dataset contains retail transaction data including:
Customer gender
Product categories
Quantity sold
Price and total sales
Transaction dates

-KEY ANALYSIS & VISUALIZATION
The dashboard includes the following insights:
 Total Sales by Gender 
 Total Sales by Category
 Quantity sold by Category
 Product Price Distribution Analysis
 Total Sales by Gender
 Cumulative Sales Growth Over Time
All visualizations are designed to support data-driven decision making.
